<h2> </h2>

Rustt (读音 / rʌsˈti /，发音类似 Rust Team) 是国内最好的 Rust 中文翻译组，由于 Rustt = Rust Translation，因此 Rustt 还是一个 Rust 翻译计划的代号，负责将国外优秀的技术文章、学习教程、新闻资讯翻译成中文。


## 订阅

- **所有**翻译文章都会存在本仓库中，大家可在右上角，🌟 收藏
- **部分**翻译文章还会同步推送微信公众号: `studyrust`
